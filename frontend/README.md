# Smart Investment Frontend (Vite + React + Ant Design + ECharts)

## 开发

1. 安装依赖
2. 启动开发服务器
3. 默认代理 `/api` 到 `http://localhost:8006`

## 脚本
- `npm run dev` 启动
- `npm run build` 构建
- `npm run preview` 预览

## 目录
- `src/pages/FutureInsights.tsx` 投资未来洞察页面
- `vite.config.ts` 代理配置

> 若后端地址不同，请修改 `vite.config.ts` 的 proxy 或在请求里使用完整 URL。
